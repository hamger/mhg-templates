# package-template
> A template of developing for javascript package.

## Construction
```
.
├── package.json  --------------------- 项目描述
├── README.md  ------------------------ 说明文件
├── build  ---------------------------- 构建配置
├── demo  ----------------------------- 演示代码
└── src  ------------------------------ 源码目录
```

## Usage
``` bash
# 安装依赖
npm install

# 本地调试
npm run dev

# 打包代码
npm run build
```
