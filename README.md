# simple-template
> A simple Webpack + babel-loader + eslint setup for ES6 with dev and prod.

## Construction
```
.
├── package.json  --------------------- 项目配置
├── README.md  ------------------------ 说明文件
├── build  ---------------------------- 构建代码文件
├── config  --------------------------- 构建配置文件
├── index.html  ----------------------- 入口页面
└── src  ------------------------------ 源码目录
```

## Usage
``` bash
# 安装依赖
npm install

# 本地调试
npm run dev
```
