# simple-template
> A simple Webpack + babel-loader setup for the develop with ES6.

## Construction
```
.
├── package.json  --------------------- 项目描述
├── README.md  ------------------------ 说明文件
├── webpack.config.js ----------------- webpack配置文件
├── index.html  ----------------------- 入口页面
└── src  ------------------------------ 源码目录
```

## Usage
``` bash
# 安装依赖
npm install

# 本地调试
npm run dev
```
