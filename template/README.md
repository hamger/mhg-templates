# react-template
这是一个react项目的模板仓库,技术栈包括react、react-router4、redux、webpack4  

## Construction
```
.
├── package.json  --------------------- 项目描述
├── README.md  ------------------------ 说明文件
├── build  ---------------------------- 构建配置
└── src  ------------------------------ 源码目录
    ├── assets  ----------------------- 资源文件
    ├── components  ------------------- 组件集合
    ├── views  ------------------------ 页面集合
    ├── store  ------------------------ redux
    ├── router.js  -------------------- 项目路由
    └── index.js  --------------------- 项目入口
├── babelrc --------------------------- babel配置
├── .eslintrc ------------------------- eslint配置
├── .gitignore ------------------------ git忽略配置
├── postcss.config.js ----------------- postcss配置
└── webpack.config.js ----------------- 判断环境
```

## Usage
``` bash
# 安装依赖
npm instal

# 启动本地调试
npm run dev

# 本地打包压缩
npm run build
```
